Chrome内核浏览器安装（Chrome 浏览器 | 搜狗，QQ，360 浏览器）
----

1. 下载项目的zip文件，在右上方有个 `Download ZIP`, 解压到本地

2. 把 .crx 文件鼠标右键拖进浏览器窗口，在弹出窗口选择 “安装” 即可完成！
